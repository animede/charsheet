# キャラクターシート作成アプリ 使い方マニュアル

1枚のキャラクター画像から、前・後ろ・左・右・斜め45度×4方向の合計8方向ビューを
自動生成し、キャラクターシートとしてまとめて閲覧・ダウンロードできるアプリです。

## 起動方法

```bash
cd /home/animede/charsheet
./venv/bin/python -m uvicorn app:app --host 0.0.0.0 --port 8600
```

ブラウザで `http://localhost:8600/` を開きます。

すでに起動しているか確認する場合:

```bash
curl -s -o /dev/null -w "%{http_code}\n" http://localhost:8600/
```
`200` が返れば稼働中です。

---

## 1. 画像をアップロードする

「1. 画像アップロード」欄に、キャラクター画像をドラッグ&ドロップするか、クリックして
ファイルを選択します。

アップロードすると自動でキャラクター検出処理が走ります。

- **1体のみの画像**の場合 → そのまま次のステップへ進めます(通常どおり)。
- **複数体(6〜8体など)が1枚に描かれたシート画像**の場合 → 「N体のキャラクターを
  検出しました。8方向生成の元にする1体を選んでください」と表示され、検出された
  各キャラクターのサムネイルが並びます。**8方向生成の元にしたい1体(通常は正面向きの図)を
  クリックして選択**してください。
  - 検出結果を使わず、アップロードした画像全体をそのまま入力にしたい場合は
    「シート全体をそのまま使う」リンクを押します。

必要であれば「seed」欄に数値を入力できます(空欄・0でランダム性のない既定シード)。
同じ入力画像・同じseedなら毎回ほぼ同じ結果が再現されます。

## 2. キャラクターシートを生成する

「キャラクターシート生成」ボタンを押すと生成が始まります。

- 初回起動後の最初の生成は、モデルの読み込みに数十秒〜数分かかります(2回目以降は
  高速です)。
- 「2. 8方向ビュー」欄に8枚のタイルが並び、完了した方向から順番に画像が表示されます
  (1方向あたり数秒程度)。
- 全8方向が完了すると、「3. キャラクターシート」欄に4×2グリッドの合成シートが
  表示されます。

生成中は他の操作(修正・背景削除・別画像での再生成)はできません(GPUを1つの処理が
占有するため)。

## 3. 個別の方向を修正する

生成後、各タイルの下にある **「修正」ボタン** を押すと入力欄が開きます。
「頭の黒い髪の毛を削除して」のように、直したい内容を日本語の文章で入力し
「適用」を押してください(Enterキーでも実行できます)。

- その方向の画像だけを対象に、指示内容を反映した再編集が行われます(数秒程度)。
- 元の姿勢や衣装、背景など指示していない部分はできる限り保持されます。
- 修正後は自動的にキャラクターシートも更新されます。

気に入る結果になるまで、同じ方向に対して何度でも修正を重ねられます。

### 元に戻す

修正した方向のタイルには **「元に戻す」ボタン** が表示されます。押すと直前の状態に
戻ります。もう一度押すと再び修正後の状態に切り替わります(行ったり来たりできる
トグル動作です。ただし保存されるのは1世代前までです)。

## 4. 背景を削除する

- 各タイルの **「背景削除」ボタン** を押すと、その方向だけ背景が透明になります。
- キャラクターシート欄の **「全方向の背景を削除」ボタン** を押すと、8方向まとめて
  背景を除去できます。
- 透明になった部分はタイル上で市松模様として表示されます(実際のファイルは透過PNG)。
- こちらも「修正」と同様、**「元に戻す」で背景ありの状態に戻せます**。

## 5. ダウンロードする

「3. キャラクターシート」欄の下に2つのボタンがあります。

| ボタン | 内容 |
|---|---|
| 一括ダウンロード (ZIP) | 8方向の画像ファイル + 合成シート画像をまとめたZIPをダウンロード |
| シート画像のみダウンロード | 合成されたキャラクターシート(1枚のPNG)のみダウンロード |

背景削除済みの方向は、ZIP内でも透過PNGのまま保存されます。

## 6. 画像を拡大して確認する

各タイルの画像をクリックすると、拡大表示(ライトボックス)が開きます。閉じるには
右上の × をクリックするか、画像の外側をクリックしてください。

## よくある操作の流れ(例)

1. キャラクター単体の画像をアップロード
2. 「キャラクターシート生成」を押して8方向が揃うのを待つ
3. 気になる方向があれば「修正」で指示を出して微調整(必要なら複数回)
4. 背景が不要ならタイルごと、または「全方向の背景を削除」で一括除去
5. 「一括ダウンロード (ZIP)」で全成果物を保存

## 複数体シートから1体を取り込む流れ(例)

1. 既存のキャラクターシート画像(4×2など)をアップロード
2. 検出されたサムネイルから使いたい1体(正面向きが理想)をクリック
3. 「キャラクターシート生成」を押すと、選んだ1体だけを元に新しい8方向シートが
   生成される

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| 「別のジョブが実行中です」と出る | 直前の生成・修正・背景削除が完了するまで待ってから再試行してください(同時に1処理のみ実行できます) |
| 生成した人物・キャラが別人になる | 通常は起きませんが、もし発生した場合はページを再読み込みしてから再生成してください。改善しない場合は管理者に連絡してください |
| ページを再読み込みしたら操作パネルが消えた | 過去のジョブ結果自体は `outputs/{job_id}/` に残っており、API経由(下記)でなら操作可能です。UI上での「過去ジョブの再開」には未対応です |
| ボタンを押しても反応しない | 画面上部にエラーメッセージが出ていないか確認してください。GPUメモリ不足などの場合はサーバーログ(`server.log`)を確認してください |

## 参考: APIを直接使う場合

curlなどからも操作できます。詳細は [README.md](../README.md) を参照してください。

```bash
# 新規生成(job_id が返る)
curl -X POST http://localhost:8600/api/generate -F "image=@character.png" -F "seed=0"

# 進捗確認
curl http://localhost:8600/api/jobs/{job_id}

# 個別方向の修正
curl -X POST http://localhost:8600/api/jobs/{job_id}/refine \
  -H "Content-Type: application/json" \
  -d '{"key": "back", "instruction": "髪の毛を削除して"}'

# 背景削除(1方向 or 全方向)
curl -X POST http://localhost:8600/api/jobs/{job_id}/remove_bg \
  -H "Content-Type: application/json" -d '{"key": "all"}'

# 元に戻す
curl -X POST http://localhost:8600/api/jobs/{job_id}/undo \
  -H "Content-Type: application/json" -d '{"key": "back"}'
```

方向キー一覧: `front`(前) / `back`(後ろ) / `left`(左) / `right`(右) /
`front_left_45`(左前45度) / `front_right_45`(右前45度) /
`back_left_45`(左後ろ45度) / `back_right_45`(右後ろ45度)
